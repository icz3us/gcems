## About the Project



**GCEF (Gordon College Event  Portal)** is a web-based platform designed to centralize and streamline campus event management.



It allows:

- **Students** to view and participate in events

- **Admin** to create evente, manage attendance

- **SuperAdmin** has full system control, including assigning roles, managing users, creating and managing events, and overseeing attendance records.



## Features



### Student Side

- View all published campus events

- Browse real-time event feed

- View event details and schedules

- Register for events

- Access dashboard for upcoming events

- Secure authentication system



### Admin Side

- Create and manage events (CRUD functionality)

- Manage attendance per event

- Publish announcements to students

- Access admin dashboard for event monitoring

  

### Super Admin Side

- Full system control

- Manage users and assign roles (Admin, Student)

- Create, update, and delete events (CRUD functionality)

- Manage attendance records across all events





### Authentication

- Secure login system for Students, Admin, and Super Admin

- Role-based access control (RBAC)

- Protected dashboards and routes per role

- Session-based authentication system
